2. Dota 2 game analysis — [Interactive dashboard in Yandex DataLens](https://datalens.yandex/j1qxzndk3mtw4?_theme=dark)  
- Match-level Dota 2 statistics: player performance, heroes, roles, and team outcomes over time.

3. Charity donors cohort analysis — [Interactive dashboard in Yandex DataLens](https://datalens.yandex/gg9eqrrgg5sk0)  
- Cohort analysis of donors for the “AIF. Dobroe Serdce” charity fund: MAU dynamics, retention by gender, average check, and LTV by region.  
- Uses PostgreSQL (via Yandex DataLens), joins, aggregations, window functions, and core product metrics (retention, LTV, ARPU-style indicators).
4. analyst-hh.sql
- SQL queries for analyzing data analyst job postings from HH.ru: data cleaning, filtering by skills, salary levels, and experience requirements.  
-  Highlights the ability to translate business questions into analytical SQL queries.
5. analyst_donor.sql
- QL queries for analyzing donations: aggregations by source, time period, and user segments.  
- Demonstrates work with joins, aggregations, window functions, and basic product metrics.
6. road_accident_analysis.ipynb
- Exploratory data analysis of road traffic accidents: distributions, time patterns, and factors such as location or conditions.  
- Includes visualizations in Python (Pandas, Seaborn, Plotly) and concise, business‑oriented insights.
7. startup_analysis.ipynb
- Analysis of startup and investment data: revenue, growth metrics, and unit economics (depending on the dataset).  
- Shows EDA, segmentation, and building metrics to support investor or product decisions.
8. yandex_delivery.pdf
- Slide deck for a Yandex Delivery PM intern test task.
- Analyzes the A–B–C intercity delivery model and proposes initiatives to grow revenue, improve profitability, and optimize capacity utilization based on the provided test data.
