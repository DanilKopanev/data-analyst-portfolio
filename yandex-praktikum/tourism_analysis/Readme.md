# [Анализ туризма и турпакетов]()

Исследовательский проект по данным туристического рынка России за 2018–2023 годы. Цель — понять динамику турпакетов и их стоимости по регионам, проверить статистические гипотезы о монетизации и росте внутреннего туризма, сформировать рекомендации для развития туристических сервисов.

## Задачи

- Подготовить и очистить данные о средствах размещения, турпотоке, турпакетах и их стоимости.
- Провести разведочный анализ (EDA): динамика по годам, топ-регионы, категории турпакетов.
- Проверить гипотезу о средней стоимости турпакетов в 2023 году (>800 млн руб.).
- Оценить статистическую значимость роста внутреннего туризма 2023 vs 2022.
- Сформулировать практические рекомендации по целевым регионам и сегментам.

## Данные и стек

**Данные:**
- `hotel.csv` — количество средств размещения и номерного фонда по типам (отели, хостелы, санатории) за 2018–2023 гг.
- `countpersonhotel.csv` — гости в 2023 году по типам (всего, россияне, иностранцы).
- `tourfirm.csv` — турпакеты по категориям и годам (всего, внутренний, выездной, въездной).
- `tourcost2023.csv` — стоимость турпакетов по категориям в 2023 году.

**Стек:** Python, Pandas, Matplotlib, SciPy, Jupyter Notebook.

## Структура

**tourism_hotel.ipynb:**
- загрузка и предобработка данных (переименование, оптимизация памяти, обработка пропусков);
- разведочный анализ: boxplot динамика размещения, топ-10 регионов по стоимости, тренды турпакетов;
- статистические тесты (одно- и двухвыборочные t-критерии);
- итоговые выводы и рекомендации.

## Ключевые выводы

- Средняя стоимость турпакетов в 2023 году статистически значимо превышает 800 млн руб. по регионам.
- Рост внутреннего туризма (турпакетов по России) в 2023 vs 2022 не подтверждён статистически.
- Рынок сильно сконцентрирован в лидирующих регионах; внутренний туризм — самый устойчивый сегмент.
- Визуализация показывает восстановление рынка после 2020 года и волатильность выездного/въездного сегментов.
